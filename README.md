# WAA-Net
Wavelet-Augmented Attention Network for Hyperspectral Unmixing

The code in this toolbox implements the "Wavelet-Augmented Attention Network for Hyperspectral Unmixing".

## System-specific notes
The code was tested in the environment of Python 3.9.16, torch 1.13.1 and CUDA 11.6. 
*(Note: Please update these versions based on the exact environment you used to run WAA-Net.)*
